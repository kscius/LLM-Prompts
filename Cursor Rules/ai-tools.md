# AI Reasoning Tools & Knowledge Management

## Available MCP Tools

### Sequential Thinking
- Use `sequentialthinking` for complex problem-solving
- Break down problems into steps
- Maintain context over multiple steps
- Filter irrelevant information

### Memory & Knowledge Graph
- `create_entities` - Create entities in knowledge graph
- `create_relations` - Create relations between entities
- `add_observations` - Add observations to entities
- `delete_entities` - Delete entities
- `delete_observations` - Delete observations
- `delete_relations` - Delete relations
- `read_graph` - Read entire knowledge graph
- `search_nodes` - Search nodes in knowledge graph
- `open_nodes` - Open specific nodes by name

### Interactive Tools
- `request_user_input` - Request user input when uncertain
- `message_complete_notification` - Notify when response is complete
- `start_intensive_chat` - Start intensive chat session
- `ask_intensive_chat` - Ask questions in intensive chat
- `stop_intensive_chat` - Stop intensive chat session

### Search & Time Tools
- `duckduckgo_web_search` - Web search for latest information
- `get_current_time` - Get current time in timezone
- `convert_time` - Convert time between timezones

## Usage Guidelines

- **Use tools proactively** to enrich understanding
- **Build contextual knowledge** using semantic graphs or related entities
- **Use search tools** to maintain fresh context
- **Never ignore available tools** - they should enhance comprehension

## Knowledge Management

- Add observations to memory MCP (tagged per project/domain)
- Update documentation after execution
- Maintain knowledge graph for project context
- Use entities and relations to track project knowledge
